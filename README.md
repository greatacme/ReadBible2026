# ReadBible2026
2026 연세대학교회 성경통독 진도 관리

개요
  성경통독 계획을 저장하고 읽기 실적을 관리한다

성경 읽기 계획
  - 일자별 성경 장절 계획을 저장
  - 26년 4월 1일 시작하여 26년 12월 31일 종료되며, 매월 계획이 한달씩 추가된다.

성경 읽기 실적
  - 계획된 장절 읽기 완료 후 화면에서 완료 체크하여 완료 여부 저장
  - 완료율 = 금일 기준 실적/계획의 백분율. 더 읽었을 경우 100% 이상도 가능

사용자 
  사용자 수 : 복수
  사용자 계정 관리: 최초 등록 시 별명을 입력 받아 해당 별병으로 기록 저장하며, 암호없이 별명으로 로그인

화면 구성
  첫화면
    - 제목 표시 : 2026년 연세대학교회 성경통독
    - 금일 기준, 전체 사용자의 통독 실적 완료율 표시
    - 완료율 최고 10명의 별명과 완료율을 막대 그래프로 표시
    - 하단에 별명 입력창과 "기록" 버튼 표시
 기록 화면
   - 개인별로 표시
   - 일자와 장절, 완료 여부 체크박스가 표시되고 수정 가능하다.
   - 체크박스 표시 후 즉시 자동 저장
